Hướng dẫn kích hoạt Đám mây

Hệ thống không thể kết nối đến máy chủ lưu trữ Firebase của bạn. Hãy đảm bảo bạn đã thực hiện đầy đủ 3 bước cấu hình sau trên trang quản trị Firebase Console:

1. Kích hoạt Anonymous Authentication:

Vào Build -> Authentication -> Sign-in method -> Thêm "Anonymous" (Ẩn danh) và gạt nút "Enable" rồi lưu lại.

2. Cấu hình Firestore Rules (Quy chế ghi đọc):

Vào Build -> Firestore Database -> Thẻ "Rules" và chỉnh sửa dòng quyền cho phép đọc/ghi công cộng:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

3. Đảm bảo kết nối Internet:

Nếu chạy file offline cục bộ, thiết bị của bạn cần kết nối Wifi/3G ổn định để gửi dữ liệu về máy chủ.
